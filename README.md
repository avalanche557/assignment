# Assignment -- Codemaster

## Run the project on local Machine

Clone the code:

###   `git clone https://github.com/avalanche557/assignment.git`

Install node modules:

### `npm install`

Run the code:

### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
